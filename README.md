## 🚀 Cómo Usar en GitHub Pages

1. Sube todos los archivos a un repositorio de GitHub
2. Ve a **Settings → Pages**
3. Selecciona la rama `main` y carpeta raíz `/`
4. Tu sitio estará disponible en: `https://tu-usuario.github.io/nombre-repo/`

### Alternativa: Netlify (recomendado)
1. Arrastra la carpeta entera a [netlify.com/drop](https://app.netlify.com/drop)
2. En segundos obtienes una URL pública gratuita
